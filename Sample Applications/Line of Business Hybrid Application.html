Line Of Business Hybrid Application

The LOB Hybrid sample application is a complete example of using the Interop Forms Toolkit.  The application consists of the following projects:

  • MyCompany_Forms - Visual Basic 6.0 application starting point, contains forms
  • MyCompany_Components - Visual Basic 6.0 business objects
  • MyCompany.Winforms.Orders - .NET assembly containing Order related forms
  • MyCompany.Winforms.Customers - .NET assembly containing Customer related forms
  • MyCompany.Components.Orders - .NET assembly containing Order related business objects
  • MyCompany.Components.Customers - .NET assembly containing Customer related business objects

 

The code for the LOB Hybrid application can be accessed from the Start Menu under Microsoft Visual Basic Power Packs | Interop Forms Toolkit 2.1 | Sample Applications.
