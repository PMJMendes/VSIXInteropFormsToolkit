Hello World Hybrid Application

The Hello World Hybrid sample application is a simple application consisting of one .NET InteropForm Library and one Visual Basic 6.0 application project. 

 

The application consists of the following projects:

  • HelloWorld - Visual Basic 6.0 application that launches .NET HelloWorld InteropForm
  • MyCompany.HelloWorld - .NET assembly containing HelloWorld InteropForm

 

The code can be accessed from the Start Menu under Microsoft Visual Basic Power Packs | Interop Forms Toolkit 2.1 | Sample Applications.
