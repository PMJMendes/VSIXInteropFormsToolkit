Microsoft InteropForm Toolkit

IInteropForm Properties

The properties of the IInteropForm interface are listed below. For a complete list of IInteropForm interface members, see the IInteropForm Members topic.

Public Instance Properties

BorderStyle Returns or sets the border style for a form. Read-only at run time.
Caption Determines the text displayed in the Form's title bar. When the form is minimized, this text is displayed below the form's icon.
Enabled Returns or sets a value that determines whether a form can respond to user-generated events.
Height Return or set the height of a form.
IsFormDisposed Indicates whether the underlying .NET Form has been disposed.
Name Returns the name used in code to identify a form. Read-only at run time.
ShowInTaskbar Returns or sets a value that determines whether a Form object appears in the Windows taskbar. Read-only at run time.
StartUpPosition Returns or sets a value specifying the position of an object when it first appears. Not available at run time.
Visible Returns or sets a value indicating whether an object is visible or hidden.
Width Returns or sets a Single containing the width of the window in twips. Read/write.
WindowState Returns or sets a value indicating the visual state of a form window at run time.
