Interop Forms Toolkit
Glossary
Hybrid Application: An application that contains both Visual Basic 6.0 and .NET forms running in a single Visual Basic 6.0 process to deliver the user experience of the application.

Interop: Within this document interop refers to cross-platform interaction between components written in .NET and Visual Basic 6.0.

Subsystem: A logical grouping of closely related functionality in an application.
